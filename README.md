Mufi kiosk를 위한 software입니다.

# Run

## 1.

### install dependencies

    yarn install

### compile typescript to javascript

    yarn run compile

### build svelte component

    yarn run build


### run server

    yarn run dev

## 2.

### the source changes, compile and run the server again.

    yarn run watch


## 3.

### develope mode on svelte

    yarn run svelte-dev
