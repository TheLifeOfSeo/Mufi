# Mufi

## setup
    yarn install
    
## production
    yarn run start
production mode로 server를 실행합니다.    
    
## dev
    yarn run dev
development mode로 server를 실행합니다.    

## compile

    yarn run compile
src하위 ts 파일들을 => dist하위 js 파일들로 컴파일합니다.

## watch
    yarn run watch
source code가 변경 될때마다 compile하여 수정된 코드로 server를 다시 실행합니다.

    yarn run watch-ts
source code가 변경될 떄마다 compile합니다.
    
## test
    yarn run test
test를 실행합니다.
